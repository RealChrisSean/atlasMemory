# Node.js Requirements for mem0-tidb

## Prerequisites
- Node.js (v18+ recommended)
- npm (comes with Node.js)
- TiDB instance (cloud or local)

## NPM Dependencies
# Install all dependencies with: npm install

### Production Dependencies
dotenv@^17.2.3          # Environment variable management
mysql2@^3.15.3          # MySQL/TiDB client library
ts-node@^10.9.2         # TypeScript execution engine
typescript@^5.9.3       # TypeScript compiler

### Development Dependencies
@types/node@^24.10.1    # Node.js type definitions
tsx@^4.20.6             # TypeScript execute (faster alternative to ts-node)

## Environment Variables Required
# Create a .env file with the following variables:
TIDB_HOST=your-tidb-host
TIDB_PORT=4000
TIDB_USER=your-username
TIDB_PASSWORD=your-password
TIDB_DATABASE=your-database

## Installation Instructions
1. Install Node.js from https://nodejs.org/ (if not already installed)
2. Clone this repository
3. Run: npm install
4. Create .env file with your TiDB credentials
5. Test the connection: npm run dev
